# KU — Kitchen Utensils by Christon

An all-in-one multi-effects plugin. Every module is off by default — toggle what you need.

## Modules

| Module | Colour | What it does |
|--------|--------|-------------|
| **EQ** | Blue | 8-band parametric with live curve display (Lo Shelf, HPF, 4× Bell, LPF, Hi Shelf) |
| **Compressor** | Teal | Analog-style compressor with knee, makeup, parallel mix |
| **Limiter** | Red | True-peak brickwall limiter with ceiling control |
| **Reverb** | Violet | Algorithmic reverb with size, damp, width, pre-delay |
| **Bass Boost** | Orange | Low-shelf boost with harmonic drive saturation |
| **Bitcrusher** | Pink | Bit depth + sample rate reduction with dither |
| **Stutter** | Lime | Gross-Beat-style gate sequencer with 8 patterns + tape effect |

## Auto Mix Mode
Press **AUTO MIX** to automatically engage EQ, Compressor, and Limiter with sensible clean-mix settings. Great for a quick starting point.

## Signal Chain Order
Input → EQ → Bass Boost → Compressor → Bitcrusher → Stutter → Reverb → Limiter → Output

## Install

**Windows:** Drop `KU.vst3` folder into `C:\Program Files\Common Files\VST3\`

**Mac:** Drop `KU.vst3` into `/Library/Audio/Plug-Ins/VST3/`

Rescan plugins in your DAW.
